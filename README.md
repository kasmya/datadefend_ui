# DataDefend - DPDPA Compliance Platform UI

## Next.js 16 Frontend

### Local Development
```
npm i
npm run dev
```
Open http://localhost:3000

### Vercel Deployment
1. Connect GitHub repo https://github.com/kasmya/datadefend_ui
2. **Framework:** Next.js (auto)
3. **Build:** `npm run build`
4. **Output:** `.next`
5. Deploy → Live!

**Build succeeded:** Static pages generated, MongoDB warnings non-blocking (static export compatible).

## Features
- Responsive design (Tailwind v4)
- Animated components
- API routes (/api/contact, /api/subscribe)
- MeitY/DPDP compliance demo
